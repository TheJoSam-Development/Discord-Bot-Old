Bot Framework Features

The Bot Framework was made for getting configured at will and be expanded by users.
But we also have added some features by default into the Framework that are making the Framework better that many other Bots.

Modular Framework

This is our Main feature of our Framework and started the idea.
You can create your own Commands and Extensions for the Bot and also can en-/disable all commands and extensions

User Register (WIP)

The Bot Framework is fitted with an User register for internal data which need an UUID, Discord User Id (DCUI), Server join date (SJD) and an Discord join date (DCJD)

Server Bot Injection (WIP)

The Server Bot Injection or SBI is an system specially created for the Framework to save the current Server status with all data present on a server. So if the Bot is Injected the Bot can do anything you want.

Styling Templates (WIP)

You can use templates created by us or others to get an special look onto the icons from Extensions/Commands.
